Spotify Dataset Analysis

Analyzing the Spotify dataset to predict song likability using a k-NN model.

Data Preprocessing

  • No missing values found.
  • Converted percentage values (e.g., danceability) into decimals.
  • Renamed columns for clarity.
  • Dataset split: Training set (60%) and validation set (40%).

Statistical Analysis

T-tests: Identified significant differences between numeric attributes (like vs. dislike).

Significant Variables:

  • Retained: danceability, tempo, valence.
  • Removed: energy, liveness, speechiness.

k-Nearest Neighbors (k-NN) Model

Implemented k-NN to predict song likability.

Found optimal k-value (7) through validation accuracy.

Credit Card Income Classification Using Naive Bayes

This study predicts individuals' income levels based on credit card usage patterns using a Naive Bayes classifier.

Data Preprocessing

  • Binning: Continuous variables were categorized using equal-width binning.

  • Encoding: Categorical variables were converted into meaningful factor levels.
