Employee Attrition Analysis: Data-Driven Insights

This project leverages exploratory data analysis (EDA) and machine learning to uncover key factors influencing employee attrition. By developing and evaluating predictive models, we provide actionable insights for HR to improve retention strategies.

Dataset & Preparation

  • 1470 employees, with key features like age, income, job satisfaction, and work-life balance.
  • Class Imbalance: 83.9% stayed, 16.1% left → required SMOTE & class weighting for better prediction.
  • Data Preprocessing: Encoded categorical variables, checked for correlations, and refined key features.

Exploratory Data Analysis (EDA) – Key Findings

  • Younger employees (avg. 33.6 years) were more likely to leave vs. 37.5 years for retained employees.
  • Higher distance from work (10.6 miles) increased attrition risk.
  • Lower salaries ($4787 vs. $6832) were linked to higher attrition.

Model Development & Evaluation

Logistic Regression

Refined by removing non-significant variables. Key attrition predictors: age, income, job satisfaction, marital status, work-life balance, and distance from home.

Performance:

  • Training Accuracy: 84.13%
  • Test Accuracy: 83.50%
  • Sensitivity: 0.0% (Failed to predict attrition cases due to class imbalance)

Random Forest Model

Hyperparameter tuning via GridSearchCV for optimization. Achieved better accuracy but still struggled with class imbalance.

Performance:

  • Training Accuracy: 86.62%
  • Test Accuracy: 84.35%
  • Sensitivity: 6.19% (Still weak at predicting attrition cases)

Key Takeaways & HR Recommendations

  • Models predict retention well but struggle with attrition cases due to class imbalance.
  • HR Strategies:
    • Improve work-life balance & career development for at-risk employees.
    • Adjust compensation & retention programs for younger, lower-income employees.
    • Implement proactive engagement policies for employees living farther away.

Conclusion

This project showcases data-driven employee retention strategies using logistic regression & random forest models. While accuracy was strong, class imbalance remains a challenge, requiring data balancing & threshold tuning for better attrition prediction. These insights empower HR teams to minimize turnover costs and enhance workforce stability.
